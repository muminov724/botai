# AI Dasturlash Boti

Telegram boti faqat dasturlash va IT sohasiga oid savollarga javob beradi. OpenAI GPT-4o-mini yordamida ishlaydi.

## O'rnatish

```bash
pip install -r requirements.txt
python bot.py
```

## Sozlash

`config.py` faylida quyidagilarni kiriting:
- `BOT_TOKEN` - Telegram Bot Token
- `OPENAI_API_KEY` - OpenAI API kaliti
